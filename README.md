# Облік товарів ювелірного магазину

Цей репозиторій містить вихідні файли курсової роботи **«Програма обліку товарів ювелірного магазину»**.

Програма створена мовою **C++** з використанням **Qt Widgets** та локальної бази даних **SQLite**. Застосунок призначений для ведення обліку ювелірних виробів: перегляду списку прикрас, додавання нових записів, редагування, видалення, пошуку за матеріалом і фільтрації за ціною.

## Основні можливості

- відображення товарів у таблиці;
- додавання нової прикраси;
- редагування даних безпосередньо в таблиці;
- видалення обраного товару;
- пошук прикрас за матеріалом;
- пошук прикрас за діапазоном ціни;
- збереження даних у базі `prykrasy.db`.

## Технології

- C++;
- Qt Widgets;
- Qt SQL;
- SQLite;
- CMake.

## Запуск проєкту

1. Встановити Qt Creator з підтримкою Qt 6 або Qt 5.
2. Відкрити файл `CMakeLists.txt` у Qt Creator.
3. Дочекатися налаштування CMake.
4. Запустити проєкт кнопкою **Run**.

Файл бази даних `prykrasy.db` додано до репозиторію як початковий приклад даних для роботи програми.
